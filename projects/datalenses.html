Data Lenses

Active Built at Numia

Overview

Analytics dashboard aimed towards showcasing specific blockchain tailored metrics the blockchain foundations want to share with their users. Currently used by the DYDX, Cosmoshub and Osmosis teams.

The Problem

Blockchain foundations want to share important protocol metrics with their community, but existing analytics tools are either too technical for general users or don't provide the customization needed to highlight the metrics that matter most to each protocol.

The Solution

Data Lenses provides customizable, public-facing dashboards that foundations can configure to showcase their most important metrics. Each dashboard is tailored to the specific needs of the protocol, presenting data in an accessible and visually appealing format for community members.

Key Features

  • Customizable metric dashboards
  • Public-facing analytics pages
  • Real-time data updates
  • Protocol-specific visualizations
  • Embeddable widgets

Tech Stack

Blockchain BigData BigQuery Tinybird React D3.js
